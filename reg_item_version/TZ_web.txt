ТЗ(Техническое задание) по проекту с использованием WEB, SQLAlchemy, Flask:
Цель: Создать сайт с текстовой игрой в жанре «Roguelike» с использованием таких технологий как WEB(интернет-пространство), SQLAlchemy(библиотека для работы с базами данных), Flask(микро-платформа для создания веб-приложений) и расширений Flask(пакетов, позволяющих настраивать более сложные функции).
Тема: Сайт с текстовой игрой в жанре Roguelike в сеттинге киберпанка.
В итоге проекта будет реализован сайт с графическим интерфейсом. В нем будут разделы, доступные всем, и разделы, доступные только зарегистрированным пользователям. Предусмотрена возможность пользователям-администраторам вносить изменения и дополнения в базу данных не напрямую из кода, а с помощью графического интерфейса.
